# Bike Website

## Project Description

This project is a responsive website dedicated to bicycles. It was built using HTML and SCSS, following the BEM methodology to ensure a clear, scalable, and maintainable code structure.

The website is designed with a strong focus on layout consistency, clean styling, and adaptability across different screen sizes.

## Features

- Responsive layout for mobile, tablet, and desktop
- SCSS architecture with modular and reusable styles
- BEM methodology for predictable and readable class naming
- Cross-browser compatible layout
- Clean and semantic HTML structure

## Technologies Used

- HTML5
- SCSS
- BEM methodology
- npm
- Git

## Project Setup

To run the project locally, follow these steps:

1. Clone the repository using Git:

   git clone <repository-url>

2. Install the required dependencies:

   npm install / npm i

3. Start the development server:

   npm run start / npm start
